## Ответ со сложными именами, где возможны имена людей и организаций

``` json
{
	"name": "КАНАФИНА НАЗЫМГУЛЬ КАНАФИЕВНА",
	"name_parts": {
		"first_name": "НАЗЫМГУЛЬ",
		"last_name": "КАНАФИНА",
		"middle_name": "КАНАФИЕВНА"
	}
}
```

Лучше чем, когда name может быть и строкой и объектом. Для органицаий в name_parts лучше вернуть все элементы с null значениями.
