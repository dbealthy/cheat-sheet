Domain Layer содержит entity, value-object, aggregate, domain-service важны для бизнеса, без конкретного сценария использования, содержит инварианты (валидации), не содержит внешних вызовов и обращений


Application Layer содержит UseCase, Repository, Unit of Work. Содержит код комманд и запросов важных для бизнеса, может образаться к внешним системам. Не содержит конкретных реализаций для репозиториев, внешних сервисов
